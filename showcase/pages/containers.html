Containers & Layouts

Layout primitives and container components for structuring WebShell applications.

Stack

Flexbox-based stacking layout for vertical or horizontal arrangements.

Vertical Stack

HTML 
<div class="ws-stack">
  <button class="ws-button ws-button--primary">Item 1</button>
  <button class="ws-button ws-button--primary">Item 2</button>
  <button class="ws-button ws-button--primary">Item 3</button>
</div>

Horizontal Stack

HTML 
<div class="ws-stack ws-stack--horizontal">
  <button class="ws-button ws-button--primary">Item 1</button>
  <button class="ws-button ws-button--primary">Item 2</button>
  <button class="ws-button ws-button--primary">Item 3</button>
</div>

Stack with Spacing Variations

HTML 
<div class="ws-stack ws-stack--horizontal ws-stack--gap-xs">...</div>
<div class="ws-stack ws-stack--horizontal ws-stack--gap-l">...</div>
