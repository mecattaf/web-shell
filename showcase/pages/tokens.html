Design Tokens

Design tokens are the visual design atoms of the design system. They define colors, typography, spacing, and other visual properties used throughout WebShell components.

Colors

Color palette including surface colors, semantic colors, and state colors.

Spacing

Consistent spacing scale for layout and component spacing.

Typography

Font sizes, weights, and families used in the design system.

Borders & Radius

Border radius values for component styling.

Elevation

Shadow values for elevation and depth.
